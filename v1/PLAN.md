# План проекта book2bert

Цель проекта — создание BERT-модели с нуля для глубокого понимания архитектуры трансформеров и получения практически полезной модели для работы с уникальным корпусом текстов (180k книг, 40 ГБ, преимущественно нехудожественная литература, включая тексты с OCR-артефактами и старой орфографией).

Проект является учебно-научным экспериментом: мы пройдем путь от сырого текста до обученной модели, с полным разбором каждого компонента.

## Технологический стек

- **Языки**: Go (обработка данных, координация), Python (NLP-сервисы, обучение модели)
- **NLP**: razdel (сегментация предложений), HuggingFace tokenizers (WordPiece), PyTorch (BERT)
- **Сервисы**: Flask, gunicorn, Docker
- **Мониторинг**: wandb / tensorboard

## Критерии успеха

1. Рабочий пайплайн от текста до датасета
2. Обученная MiniBERT на 50 книгах с адекватным loss
3. Визуализированные attention patterns, подтверждающие понимание структуры текста
4. Открытый репозиторий с воспроизводимыми шагами
5. Документированные выводы и инсайты

## Примечания

- Все сгенерированные данные (processed/, models/) добавлены в .gitignore
- 50 чистых книг будут добавлены как примеры и для начального обучения
- Код пишется с учетом возможности масштабирования на полный корпус (40 ГБ)
