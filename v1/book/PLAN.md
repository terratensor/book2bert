# План книги "Создание BERT с нуля"

## Целевой корпус
- **Militera** (военно-исторический подраздел)
- 3.6 млн фрагментов, ~4 ГБ
- Военная терминология, OCR-артефакты, старая орфография

## Структура

### Глава 1: Зачем создавать BERT с нуля?
- Мотивация (учебная + практическая)
- Описание militera корпуса
- Почему готовые модели не подходят

### Глава 2: Подготовка данных
- Сегментация предложений (razdel + HTTP-сервис)
- Фильтрация CJK/тайских символов
- Разбивка по `\n` для списков

### Глава 3: Токенизация
- WordPiece: теория и практика
- Обучение токенизатора на militera
- Ускорение через Go-утилиту

### Глава 4: Сборка датасета
- Формат JSONL для BERT
- Проблема двойного [CLS]/[SEP]
- Проблема потери [SEP] между предложениями
- Батчевая токенизация (17 минут вместо 6-8 часов)

### Глава 5: Архитектура BERT
- Multi-Head Self-Attention
- Positional Encoding
- Transformer Block
- BERTForMLM

### Глава 6: Обучение
- Streaming-загрузка данных
- Mixed precision (fp16)
- Стратегия чекпоинтов
- OneCycleLR scheduler

### Глава 7: Результаты
- Эпоха 1: val_loss 3.43, perplexity 30.8
- Эпоха 2: val_loss 2.96, perplexity 19.3
- MLM предсказания: осмысленные слова

### Глава 8: Выводы
- Качество данных важнее архитектуры
- Токенизатор на большом корпусе — ключевой фактор
- Python + Go: синергия для больших данных

### Приложения
- А: Все команды для воспроизведения
- Б: Сравнение моделей (50 книг vs militera)
- В: Словарь терминов

## Реальные цифры militera корпуса
- Исходные сжатые файлы: 3.6 GB (.txt.gz)
- Распакованный текст: ~15-20 GB
- Корпус после фильтрации: 14 GB
- Предложений: 93,124,036
- Файлов: 11,361
- Train примеры для BERT: 3,931,780
- Val примеры: 223,044
