# Создание BERT с нуля

## Практическое путешествие от идеи до обученной модели

---

### Об этой книге

Эта книга родилась из реального эксперимента по созданию BERT-модели для военно-исторического корпуса. Мы не просто вызывали `from transformers import ...`, а прошли весь путь: от сырых текстов в кодировке Windows-1251 до обученной модели, которая понимает военную терминологию и философские концепты.

**Целевая аудитория:** инженеры и исследователи, знакомые с ML, но не углублявшиеся в архитектуру трансформеров.

**Что вы узнаете:**
- Как подготовить данные для BERT (самый важный этап)
- Как работает токенизация WordPiece
- Как собрать датасет для BERT
- Как устроен BERT "под капотом"
- Как оптимизировать обучение (streaming, fp16, gradient accumulation)
- Как анализировать результаты (MLM, attention, [CLS] эмбеддинги)

**Технологический стек:** Go (обработка данных), Python (NLP, обучение), PyTorch, razdel, HuggingFace tokenizers.

**Исходный код:** [github.com/terratensor/book2bert](https://github.com/terratensor/book2bert)
