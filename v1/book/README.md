# book2bert: Создание BERT с нуля

## Краткое описание

Это книга о практическом создании BERT-модели для военно-исторического корпуса. Мы прошли весь путь: от сырых текстов в кодировке Windows-1251 до обученной модели, которая понимает военную терминологию и философские концепты.

## Для кого эта книга

- Инженеры, знакомые с ML, но не углублявшиеся в архитектуру трансформеров
- Разработчики, работающие с NLP на Go и Python
- Все, кто хочет понять BERT "изнутри"

## Что вы найдете в книге

| Раздел | Описание |
|--------|----------|
| **Главы 1-2** | Мотивация и подготовка данных (самый важный этап) |
| **Глава 3** | Токенизация WordPiece (от субвордов к смыслу) |
| **Глава 4** | Сборка датасета для BERT (оптимизация 6-8 часов → 17 минут) |
| **Глава 5** | Архитектура BERT (реализация на PyTorch) |
| **Глава 6** | Обучение (streaming, fp16, чекпоинты) |
| **Глава 7** | Результаты (MLM, attention, [CLS] эмбеддинги) |
| **Глава 8** | Выводы и уроки |
| **Приложения** | Команды, сравнение моделей, словарь терминов |

## Технологический стек

- **Go**: обработка больших объемов данных, параллелизм
- **Python**: NLP-сервисы (razdel), обучение модели (PyTorch)
- **HTTP/gRPC**: коммуникация между Go и Python

## Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/terratensor/book2bert.git
cd book2bert

# Установить зависимости
pip install -r requirements.txt

# Запустить сервис сегментации
cd services/segmenter
gunicorn -w 8 -b 0.0.0.0:8090 app:app

# Обработать корпус
go run cmd/process-corpus/main.go \
    --corpus /path/to/corpus \
    --output data/processed/sentences \
    --segmenter http://localhost:8090 \
    --workers 10
