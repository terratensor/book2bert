# Эксперимент book2bert

## Цель
Создать BERT-модель с нуля для понимания архитектуры трансформеров и получения полезной модели для работы с корпусом нехудожественной литературы.

## Фаза 1: Подготовка данных (пробный запуск)

### 1.1 Исходные данные
- 50 книг философской тематики (27.6 МБ)
- Кодировка: Windows-1251 (требуется конвертация в UTF-8)
- Формат: TXT (после конвертации из DOCX)

### 1.2 Сегментация предложений
- Использован сервис на Flask + razdel
- Порт: 8090
- Результат: 152,444 предложения

### 1.3 Обучение токенизатора WordPiece
- **Проблема:** vocab_size=30000 на 50 книгах → в словаре только субворды
- Целые слова (учение, граница, материя) отсутствуют
- **Вывод:** нужен vocab_size=50000 или больше данных

### 1.4 Создание датасета для BERT
- Группировка предложений по 512 токенов
- Результат: 11,528 train + 576 val примеров

### 1.5 Обучение BERT-tiny (22M параметров)
- 50 эпох, 102 секунды на эпоху
- Лучший val_loss: 6.97
- MLM предсказывает знаки препинания, не смысл
- **Причина:** модель учится предсказывать субворды (окончания)
