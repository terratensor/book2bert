## Сборка датасета полный датасет, часть 10-15%

### 1. Полный датасет (как раньше)

```bash
python scripts/build_dataset.py \
    --sentences-dir /mnt/archive/book2bert/data/processed/sentences_full \
    --tokenizer-path data/processed/tokenizer_full \
    --output-dir /mnt/archive/book2bert/data/processed/dataset_full \
    --max-length 512 \
    --val-split 0.05 \
    --batch-size 2000
```

### 2. Датасет из 10% предложений (для tiny/small экспериментов)

```bash
python scripts/build_dataset.py \
```

---

## Что изменилось

| Изменение | Описание |
|-----------|----------|
| **Новый параметр** | `--sample-ratio` — доля предложений для включения (0.0-1.0) |
| **Новый параметр** | `--sample-seed` — seed для воспроизводимости |
| **Логика выборки** | В `stream_sentences_by_book` каждое предложение включается с вероятностью `sample_ratio` |
| **Обработка книг** | Книги с пустыми наборами предложений пропускаются |

---

## Преимущества

| Аспект | Преимущество |
|--------|--------------|
| **Гибкость** | Можно быстро собрать датасет любого размера без пересборки исходного корпуса |
| **Воспроизводимость** | Фиксированный seed даёт одинаковую выборку при повторных запусках |
| **Скорость** | Выборка происходит на лету, без дополнительных проходов |
| **Совместимость** | Логика группировки и токенизации не изменилась |
