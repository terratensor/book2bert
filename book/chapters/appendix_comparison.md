# Приложение Б: Сравнение моделей

## Эксперимент 1: 50 книг (плохой токенизатор)
- **Данные:** 50 философских книг (27 МБ, 152k предложений)
- **Токенизатор:** vocab_size=30000 (обучен на 50 книгах)
- **Модель:** BERT-tiny (22M параметров)
- **Результат:** val_loss ~7.8, предсказывает знаки препинания

## Эксперимент 2: militera v2
- **Данные:** 11,361 книга (14 ГБ текста, 93M предложений)
- **Токенизатор:** vocab_size=50000 (обучен на militera)
- **Модель:** BERT-tiny (29.9M параметров)
- **Особенности:** без `[SEP]` между предложениями, с CJK фильтрацией
- **Результат (эпоха 2):** val_loss 2.96, perplexity 19.3

## Эксперимент 3: militera v3
- **Данные:** те же, что в v2
- **Токенизатор:** vocab_size=50000
- **Модель:** BERT-tiny (29.9M параметров)
- **Особенности:** `[SEP]` между предложениями, CJK фильтрация, оптимизированная сборка
- **Результат (эпоха 1):** val_loss 3.41, perplexity 30.21
- **Ожидание (эпоха 5):** val_loss ~2.80-2.90, perplexity ~16-17
